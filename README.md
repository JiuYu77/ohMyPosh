# Nerd Fonts
[Nerd Fonts 官网](https://www.nerdfonts.com/)

注意：
* 安装Nerd Fonts后，若仍然显示乱码，可尝试重启计算机。
* Nerd Font Libraries 包含多种字体，每个字体可能会有不同的符号；因此，选择的字体不正确，也可能会导致乱码。
